# 霓虹小行星 - 颜色设计规范

## 概述

本文档定义了《霓虹小行星（Neon Aster）》的完整颜色规范体系。所有游戏元素必须严格遵守此规范，以确保统一的霓虹赛博朋克视觉风格。

**设计理念**：基于游戏Logo的霓虹蓝-粉渐变风格，营造充满活力的赛博朋克氛围。

---

## 实体颜色规范

### 玩家系统

#### 玩家飞机
- **主色**：`0x00b8ff`（霓虹蓝）
- **发光**：同色，alpha 0.6-0.8
- **用途**：主角战机，代表玩家控制的友方单位
- **不可更改**：玩家颜色必须始终为蓝色系，确保识别度

#### 玩家子弹
- **主色**：`0x88ddff`（浅蓝）
- **发光**：同色，alpha 0.5
- **用途**：主武器发射的投射物
- **原则**：比玩家飞机稍浅，保持色系一致

#### 僚机
- **主色**：`0x00ddff`（霓虹青）
- **发光**：同色，alpha 0.6
- **用途**：玩家召唤的辅助单位
- **原则**：青蓝色系变体，区别于主角但保持友方识别

#### 僚机子弹
- **主色**：`0x66ccff`（淡青）
- **发光**：同色，alpha 0.4
- **用途**：僚机武器发射的投射物

---

### 敌人系统（按形状分类）

#### 六边形敌人
- **主色**：`0x00b8ff`（霓虹蓝）
- **发光**：同色，alpha 0.5-0.7
- **用途**：六边形形状的敌人（慢速、肉盾型）
- **示例**：`hex_basic`

#### 三角形/箭头敌人
- **主色**：`0xff0088`（霓虹粉）
- **发光**：同色，alpha 0.5-0.7
- **用途**：三角形/箭头形状的敌人（快速、敏捷型）
- **示例**：`arrow_fast`

#### 方形/菱形敌人
- **主色**：`0xaa44ff`（霓虹紫）
- **发光**：同色，alpha 0.5-0.7
- **用途**：方形/菱形形状的敌人（平衡型、特殊攻击）

#### 圆形敌人
- **主色**：`0x00ddff`（霓虹青）
- **发光**：同色，alpha 0.5-0.7
- **用途**：圆形形状的敌人（辅助型、召唤型）

---

### Boss系统

#### Boss主体
- **主色**：`0xff0088`（霓虹粉）
- **发光**：`0xaa44ff`（霓虹紫），alpha 0.4-0.6（双层发光）
- **用途**：关底Boss的主体颜色
- **原则**：粉+紫双层，营造强大威胁感

#### Boss子弹/攻击
- **主色**：`0xff44aa`（粉紫混合）
- **发光**：同色，alpha 0.5
- **用途**：Boss发射的投射物和特殊攻击
- **原则**：粉紫混合色，区别于普通敌人

#### Boss阶段变化（可选）
- **阶段1**：`0xff0088`（霓虹粉）
- **阶段2**：`0xaa44ff`（霓虹紫）- 主色转变
- **阶段3**：`0xff0088` + `0xaa44ff` 快速闪烁

---

### 拾取物与奖励

#### 经验碎片
- **主色**：`0xffdd44`（霓虹金）
- **发光**：同色，alpha 0.6-0.8
- **用途**：击杀敌人掉落的经验值
- **原则**：金色象征收益，高识别度

#### 治疗包
- **主色**：`0x00ff88`（霓虹绿）
- **发光**：同色，alpha 0.6
- **用途**：恢复生命值的道具
- **原则**：绿色代表治疗/正面，游戏通用语言

#### 特殊道具/能量包
- **主色**：`0xaa44ff`（霓虹紫）
- **发光**：同色，alpha 0.7
- **用途**：稀有道具、能量补充
- **原则**：紫色象征稀有/特殊

## UI颜色规范

### 按钮系统

#### 主要按钮
- **边框**：`0x00b8ff`（霓虹蓝）到 `0xff0088`（霓虹粉）水平渐变
- **发光**：双层 - 蓝色（alpha 0.3-0.5）+ 粉色（alpha 0.2-0.35）
- **文字**：白色 `0xffffff` 或渐变色
- **背景**：深色半透明 `0x1a1a35` alpha 0.9

#### 次要按钮
- **边框**：`0x00b8ff`（霓虹蓝）纯色
- **发光**：单层蓝色，alpha 0.3
- **文字**：白色 `0xffffff`
- **背景**：深色半透明 `0x0f0f20` alpha 0.9

#### 禁用/不可用按钮
- **边框**：`0x444444`（灰色）
- **文字**：`0x666666`（浅灰）
- **背景**：深色半透明 `0x0a0a0a` alpha 0.5

### 进度条

#### 经验条
- **填充**：`0x00b8ff`（蓝）到 `0x00ddff`（青）水平渐变
- **背景**：`0x1a1a2e` alpha 0.6
- **边框**：`0x00b8ff` alpha 0.8

#### 生命值条
- **填充**：`0x00ff88`（绿）到 `0x00dd77`（深绿）渐变
- **低血量**：`0xff8844`（橙）到 `0xff0088`（粉）渐变（<30%）
- **背景**：`0x2a1a1e` alpha 0.6
- **边框**：根据状态变化

#### Boss血条
- **填充**：`0xff0088`（粉）到 `0xaa44ff`（紫）渐变
- **背景**：`0x2a1a2e` alpha 0.8
- **边框**：`0xff0088` alpha 1.0

### 文字颜色

#### 伤害数字
- **颜色**：`0xff0088`（霓虹粉）
- **发光**：同色，alpha 0.5
- **描边**：黑色，宽度2px

#### 治疗数字
- **颜色**：`0x00ff88`（霓虹绿）
- **发光**：同色，alpha 0.5
- **描边**：黑色，宽度2px

#### 经验/金币数字
- **颜色**：`0xffdd44`（霓虹金）
- **发光**：同色，alpha 0.5
- **描边**：黑色，宽度2px

#### 一般UI文字
- **主要文字**：`0xffffff`（白色）
- **次要文字**：`0xaaaaaa`（浅灰）
- **标题文字**：`0x00b8ff`（霓虹蓝）或渐变

### 警告与提示

#### 警告信息
- **颜色**：`0xff8844`（霓虹橙）
- **发光**：同色，alpha 0.6
- **用途**：中等危险提示

#### 严重警告
- **颜色**：`0xff0088`（霓虹粉）
- **发光**：同色，alpha 0.7
- **用途**：高危险、紧急提示

#### 正面提示
- **颜色**：`0x00ff88`（霓虹绿）
- **发光**：同色，alpha 0.5
- **用途**：成功、获得奖励

## 特效颜色规范

### 爆炸效果

#### 小型爆炸（敌人死亡）
- **核心**：`0xff8844`（霓虹橙）
- **外围粒子**：继承敌人颜色
- **发光**：核心色，alpha 0.8

#### 大型爆炸（Boss/玩家死亡）
- **核心**：`0xff8844`（霓虹橙）
- **中层**：`0xff0088`（霓虹粉）
- **外层**：`0xaa44ff`（霓虹紫）
- **粒子数量**：更多，持续时间更长

### 粒子效果

#### 引擎尾焰（玩家）
- **颜色**：`0x00b8ff`（霓虹蓝）渐变到透明
- **发光**：强烈，alpha 0.7-0.9

#### 能量充能效果
- **颜色**：`0x00b8ff`（蓝）+ `0xaa44ff`（紫）交替闪烁
- **发光**：强烈，alpha 0.8

#### 受击闪光
- **玩家受击**：白色 `0xffffff` 闪烁
- **敌人受击**：白色 `0xffffff` 闪烁
- **持续时间**：0.08秒

## 使用规则与最佳实践

### 基本原则

1. **友敌分明**
   - 蓝色系 = 友方（玩家、僚机、友好元素）
   - 粉/紫/橙色系 = 敌方（敌人、Boss、危险元素）
   - 金/绿色系 = 中立收益（奖励、治疗）

2. **形状颜色绑定**
   - 敌人颜色由形状决定，保持一致性
   - 同一形状的敌人必须使用相同颜色
   - 不同难度可调整亮度，但色相不变

3. **发光效果**
   - 所有霓虹元素必须有发光效果
   - 发光颜色与主色相同
   - 发光透明度：0.3-0.8之间
   - 重要元素发光更强

4. **对比度要求**
   - 确保所有UI文字与背景有足够对比
   - 玩家与敌人颜色必须易于区分
   - 重要信息使用高饱和度颜色

### 颜色组合建议

#### 双色边框（推荐用于重要UI）
```
边框层1: 0x00b8ff (霓虹蓝) alpha 1.0
边框层2: 0xff0088 (霓虹粉) alpha 0.6
发光层1: 0x00b8ff alpha 0.4
发光层2: 0xff0088 alpha 0.3
```

#### 渐变进度条
```
起始色: 0x00b8ff (霓虹蓝)
结束色: 0x00ddff (霓虹青)
方向: 水平从左到右
```

#### Boss多层发光
```
主体: 0xff0088 (霓虹粉)
内发光: 0xff0088 alpha 0.6
外发光: 0xaa44ff alpha 0.4
```

### 禁止事项

❌ **禁止使用的颜色**
- 纯红色 `0xff0000` - 过于刺眼
- 纯绿色 `0x00ff00` - 与治疗色冲突
- 黄绿色系 - 与霓虹风格不符
- 棕色、土色系 - 不符合赛博朋克风格

❌ **禁止的操作**
- 不要随意改变玩家飞机颜色
- 不要混淆友敌颜色系统
- 不要使用低饱和度颜色（灰色除外）
- 不要在同一形状敌人上使用多种颜色
- 不要省略发光效果

✅ **推荐做法**
- 使用渐变增加层次感
- 保持色系一致性
- 重要元素加强发光
- 使用颜色编码传达信息

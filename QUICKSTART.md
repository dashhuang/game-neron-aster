# 霓虹小行星 - 快速开始

## 🚀 运行游戏

### 开发模式

```bash
# 安装依赖（如果还没安装）
npm install

# 启动开发服务器
npm run dev
```

游戏将在浏览器中自动打开，访问 `http://localhost:3000`

### 生产构建

```bash
# 构建生产版本
npm run build

# 预览构建结果
npm run preview
```

## 🎮 操作说明

### PC 端
- **移动**: `W/A/S/D` 或 `方向键`
- **射击**: 自动射击
- **暂停**: `ESC`（未实现）

### 移动端
- **移动**: 触摸屏幕任意位置，作为虚拟摇杆中心
- **射击**: 自动射击

## 🎨 美术风格

游戏采用统一的霓虹赛博朋克风格（详见 [颜色设计规范](docs/COLOR_DESIGN.md)）：
- **背景**: 深色星空 + 噪点
- **玩家**: 霓虹蓝箭头飞船（0x00b8ff）
- **敌人**: 按形状分色（六边形=蓝、三角形=粉、方形=紫）
- **僚机**: 霓虹青小箭头（0x00ddff）
- **经验**: 霓虹金碎片（0xffdd44）
- **UI**: 霓虹蓝主色调（0x00b8ff）

## 📊 性能指标

- **目标帧率**: 60 FPS
- **最低帧率**: 45 FPS（中低端移动设备）
- **实体限制**:
  - 敌人: ≤ 200
  - 子弹: ≤ 50（玩家/敌人/僚机共用）
  - 特效: ≤ 60

## 🔧 技术栈

- **渲染**: PixiJS v8
- **语言**: TypeScript
- **构建**: Vite
- **架构**: 自研简化 ECS

## 📁 项目结构

```
src/
  ├── core/           # 核心引擎（ECS、对象池）
  ├── components/     # 数据组件
  ├── systems/        # 逻辑系统
  ├── entities/       # 实体工厂
  ├── graphics/       # 霓虹渲染
  ├── config/         # 配置常量
  └── main.ts         # 入口文件
```

## 🎯 核心玩法

1. **移动** - 单手操控，躲避敌人
2. **射击** - 自动射击，击杀敌人
3. **拾取** - 磁吸经验碎片
4. **升级** - 累积经验提升等级

## 🐛 调试

打开浏览器控制台（F12）可以看到：
- FPS 显示在游戏右上角
- 控制台日志输出游戏事件

## 📱 移动端测试

使用 Chrome 开发者工具的设备模拟器：
1. 按 `F12` 打开开发者工具
2. 点击 `Toggle device toolbar` (Ctrl+Shift+M)
3. 选择移动设备（如 iPhone 12 Pro）
4. 刷新页面测试

## 🎨 自定义配置

修改 `src/config/constants.ts` 可以调整：
- 颜色方案
- 游戏数值（血量、伤害、速度等）
- 性能限制

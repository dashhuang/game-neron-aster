## 🎨 代码示例

### 添加新敌人（3 行配置）

**编辑**: `public/data/enemies/enemies.json`
```json
{
  "id": "elite_purple",
  "hp": 150,
  "speed": 90,
  "color": 11158783,
  "shape": "hexagon",
  "aiType": "tracking"
}
```

**使用**: 自动生效，无需改代码！

### 添加新武器（3 行配置）

**编辑**: `public/data/weapons/weapons.json`
```json
{
  "id": "laser_pierce",
  "damage": 15,
  "fireRate": 5.0,
  "pierce": 3
}
```

**使用**: 修改玩家 `startWeapon` 即可！

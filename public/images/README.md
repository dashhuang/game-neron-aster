- 格式：PNG（支持透明背景）
- 推荐尺寸：1200x600 像素或更高
- 风格：霓虹赛博朋克风格
- 内容：游戏标题"霓虹小行星 / NEON ASTER"

- 加载时会自动缩放以适应屏幕

- Logo大幅放大以突出游戏品牌（是原设计的近3倍）
